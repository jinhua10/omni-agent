# React 组件

按功能模块组织的 React 组件。

## 目录结构

```
components/
├── common/          # 通用组件（Button, Modal, Toast 等）
├── layout/          # 布局组件（Header, Footer, Sidebar）
├── qa/              # 问答模块
├── document/        # 文档管理模块
├── statistics/      # 统计模块
├── feedback/        # 反馈系统
├── role/            # 角色管理（Phase 8）
├── wish/            # 愿望单系统（Phase 8）
├── collaboration/   # 协作网络（Phase 9）
├── ai-service/      # AI 服务市场（Phase 9）
├── user/            # 个人中心（Phase 9）
└── admin/           # 系统管理（Phase 9）
```

## 组件规范

1. ✅ 使用 JSX 格式
2. ✅ 文件名采用 PascalCase（如 `Button.jsx`）
3. ✅ 样式独立到 CSS 文件
4. ✅ 添加中英文注释
