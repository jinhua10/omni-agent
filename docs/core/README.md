## 📝 文档规范

### 文档命名

- 使用 `UPPERCASE_WITH_UNDERSCORES.md` 格式
- 英文版文档添加 `_EN` 后缀
- 避免使用特殊字符和空格

### 文档结构

所有文档应包含：

```markdown
# 文档标题

> **版本：** x.x.x  
> **更新时间：** YYYY-MM-DD  
> **状态：** ✅/⚠️/❌

## 📋 目录
...

## 内容
...

---

**文档维护者：** OmniAgent Team  
**最后更新：** YYYY-MM-DD
```

### 文档状态

- ✅ **生产就绪** - 文档完整、准确、已审核
- ⚠️ **进行中** - 文档正在完善
- ❌ **已废弃** - 文档已过时，不再维护
